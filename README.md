# 🚀 Project Overview

This platform enables restaurant customers to scan QR codes placed on tables, browse digital menus, place food orders, request waiter assistance, and complete payments directly from their devices.

The system centralizes restaurant operations through dedicated dashboards for:

- 👤 Customers
- 👨‍🍳 Kitchen Staff
- 🧑‍💼 Waiters
- ⚙️ Administrators

Designed for scalability, operational efficiency, and modern restaurant workflows.
